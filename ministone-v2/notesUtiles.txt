URL du tuto : https://dzone.com/articles/bootiful-development-with-spring-boot-and-angular

Commande pour lancer le serveur Spring : mvn clean install (à partir de la racine server)
Pour le quitter, il ne faut pas oublier de fermer le processus "Java(TM) Platform SE Binary" dans les tâches de fond

Commande pour lancer le serveur Angular : ng serve (à la racine du dossier client)
ne pas oublier de faire un "npm i" dans /client lors de la première copie
